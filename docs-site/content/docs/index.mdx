**MarzbanSDK** is a fully typed client library for the [Marzban](https://github.com/Gozargah/Marzban)
API. It runs in both **Node.js** and the **browser**, giving you a clean,
strongly-typed interface to Marzban's full feature set — real-time WebSocket
logs, automatic token refresh, and robust retry handling included.
